# socket.io-with-io.js
simple example for use io.js and express and socket.io say hello word.

## First step
npm install

## Second step
bower install

## Final
* Run `iojs app.js`
* Open bowser `localhost:3000`
* Open incognito window `localhost:3000`
* Type some word and check out the 2 browser
* Try it :D
